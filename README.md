# Lift-reloaded
the Reloaded version of Lift plugin in minecraft, recently support newest version 1.14.3!!!

- Build this file by the following command:

  - `mvn clean`
  - ```mvn install```
  
  [Optional]
  - `mv target/Lift-VER.jar /path/to/server/plugins`
  - `spigot command reload`
  
  [Requirements]
  - Spigot
  - Maven
